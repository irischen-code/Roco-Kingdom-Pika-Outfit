更新日志 皮卡随机商店 · 规划助手
v1.2 当前版本 2026 年 6 月
🚀 新增与特性
移动端全量适配：完成 Mobile 视图重构，针对手机端网页进行流畅度优化，并将关键弹窗统一升级为"底部抽屉式"交互。
套装数据补全：正式解锁此前挂起的 7 部件套装模块（含野外考察队、魔法嘉年华），全站收录时装总数同步更新至 37 套。
⚙️ 算法与计算优化
计算粒度精细化：将待入池策略的概率算法由"整套判定"优化为"单件部件维度"，计算结果更契合实际翻牌场景。
机制联动修正：优化多套合并计算逻辑，目标套装中若包含已持有部件，系统将在模拟时自动将其作为"已排除资产"进行加权扣除。
🎨 界面与体验升级
组件交互修正：修正"整套 / 部分"持有模式切换时的前端联动逻辑，处于"已持有"状态的部件将强制转为灰显且不可二次勾选。
排版错位微调：精简顶部导航栏布局，提升高分辨率屏幕下的信息视觉密度。
🔧 缺陷修复
修复 macOS 及移动端环境下，页面纵向无法顺畅滚动的弹性回弹问题。
修复因浏览器原生滚动条轨道占位导致的顶部导航栏右侧留缝、侧边栏横向错位缺陷。
v1.1 GitHub 首发版本
核心预测上线：支持入池倒计时悬浮面板展现，并引入"欧皇 / 正常 / 非酋"三档集齐日期概率预测模拟（3000 次随机模拟）。
策略覆盖全面：支持部件清池策略建议及多套合并计算，每日许愿星收入支持玩家手动覆盖调试。
数据基础构建：上线亮色主题二次元 UI，采用顶部双页签导航及三步引导交互，首发录入 37 套时装基础数据。
v1.0 内部 Demo 版本
跑通基础概率与许愿星规划核心算法模型，支持单套目标的天数精算。
确立部件、套装卡、混合策略下 P50 / P90 的天数计算公式，金色 195 星 / 紫色 98 星。
