# 皮卡随机商店 · 规划助手

> 洛克王国随机商店时装规划工具 —— 输入你有什么、想要什么，输出多久能凑齐、怎么规划最优。

🔗 **在线使用：[irischen-code.github.io/Roco-Kingdom-Pika-Outfit](https://irischen-code.github.io/Roco-Kingdom-Pika-Outfit/)**

---

## 功能介绍

- **命中概率计算**：基于当前卡池大小，精确计算今日翻到目标部件的概率
- **集齐周期预测**：蒙特卡洛模拟（3000次）给出欧皇 / 正常 / 非酋三档集齐天数与具体日期
- **许愿星收支规划**：计算目标总花费，判断现有星数是否够用、还需攒几天
- **清池策略建议**：识别卡池中低价非目标部件，推荐提前买断以压缩卡池、提升命中率
- **待入池套装估算**：目标套装尚未入池时，按入池日期做未来卡池快照，提前估算入池后概率
- **入池倒计时面板**：全页面常驻悬浮面板，实时显示下一批套装的精确入池倒计时
- **卡池总览**：全部专题套装一览，区分已入池 / 待入池 / 挂起状态

## 数据说明

- 内置数据为时点快照，当前收录 **37 套**时装（截至魔法嘉年华专题）
- 卡池状态根据系统实时日期自动过滤，未入池套装不参与概率计算
- 每两周随游戏更新迭代一次数据

## 技术栈

单文件纯原生实现，无任何框架依赖。

- HTML / CSS / JavaScript（Vanilla）
- 蒙特卡洛概率引擎（3000次模拟）
- 完整适配 PC 端与移动端 H5
